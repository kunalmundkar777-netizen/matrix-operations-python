# Matrix Operations using Python and NumPy

## Project Overview
This project performs fundamental matrix operations using Python and NumPy.

It is a menu-driven Python application implementing important linear algebra concepts.

---

## Features
- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Transpose
- Determinant Calculation
- Inverse of Matrix
- Rank of Matrix

---

## Technologies Used
- Python
- NumPy

- Matrix Operations
- Numerical Computation
